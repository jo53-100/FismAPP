🎓 API de Certificados FCFM

Esta página demuestra cómo usar los endpoints públicos de la API de certificados sin necesidad de autenticación.

📋 Información de la API

👥 Lista de Profesores

Obtén la lista de profesores disponibles en el sistema.

📄 Plantillas Disponibles

Obtén la lista de plantillas de certificado disponibles.

📜 Solicitar Certificado

Genera un nuevo certificado para un profesor.

🔍 Verificar Certificado

Verifica la autenticidad de un certificado usando su código de verificación.

📚 Ejemplos de Uso

1. Solicitar un Certificado (cURL)

curl -X POST http://localhost:8000/api/certificates/request-public/ \ -H "Content-Type: application/json" \ -d '{ "id_docente": "100224988", "destinatario": "A QUIEN CORRESPONDA", "incluir_qr": true }'

2. Verificar un Certificado (cURL)

curl "http://localhost:8000/api/certificates/verify-public/?code=VERIFICATION_CODE"

3. Listar Profesores (cURL)

curl "http://localhost:8000/api/certificates/professors-public/"

4. Listar Plantillas (cURL)

curl "http://localhost:8000/api/certificates/templates-public/"
